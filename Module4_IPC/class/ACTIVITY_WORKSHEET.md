## 📞 Quick Reference

### Compile & Run
```bash
gcc -Wall -Wextra -o program program.c
./program
```

### Check IPC Resources
```bash
ipcs -a              # List all
ipcs -q              # Message queues
ipcs -s              # Semaphores
```

### Clean Up IPC
```bash
ipcrm -q <queue_id>
ipcrm -s <sem_id>
make clean
```

### Debug
```bash
valgrind ./program   # Check memory leaks
gdb ./program        # Debugger
strace ./program     # Trace system calls
```
